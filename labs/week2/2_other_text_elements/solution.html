Metaphors are important to our understanding because they create concrete examples that we can use to understand abstract processes.

The desktop metaphor is characterized by windows and hierarchy of folders. A computer screen contains many windows, and each window hosts an application. Users can open, resize and minimize a window, move its content by scrolling, find actions to perform on the drop-down menu, and move the window around to see other windows. Beneath all the windows is the desktop folder. A folder contains references to other folders and files, which are documents for storing information; users can copy, paste and move around files and folders to organize them.
— from Computer Metaphors by Zhixuan Lai
